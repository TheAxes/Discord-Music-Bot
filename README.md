# Discord-Music-Bot
A Cool Music Bot In Python!



### Installation 
# Skip If You Want to run on replit 


1. Clone 
```
git clone https://github.com/TheAxes/Discord-Music-Bot.git
```
2. Manage Configs
```
Enter Activity Name Or Prefix in Config.json,
Put bot token in env
token key name = token
```
3. Run Python File
```
python main.py
```
